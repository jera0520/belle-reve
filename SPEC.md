# Belle Rêve - Project Specification

## 프로젝트 개요

**프로젝트명**: Belle Rêve (벨 레브)  
**의미**: 프랑스어로 "아름다운 꿈"  
**설명**: K-뷰티의 검증된 원료·기술을 기반으로 각 국가의 문화 정체성(Cultural Identity)을 반영하여 국가별로 '나만의 프리미엄 에디션'을 공동 제작(Co-creation)하는 화장품 브랜드

**창업자**:
- 이세라 (Lee Sera) - 공동 창업자 & CEO
- 김진현 (Kim Jinhyeon) - 공동 창업자 & CTO

---

## 1. 사업 개요

### 핵심 컨셉
- K-뷰티의 검증된 원료·기술 기반
- 국가별 Cultural Identity + 고객 참여형 디자인/용기/향 조합
- 고객 피드백 기반 제품 개발로 높은 초기 충성도 확보

### 브랜드 포지셔닝
프리미엄 K-뷰티 × 글로벌 문화 융합 브랜드

---

## 2. 문제 정의

### 글로벌 고객 요구사항

#### 일본
- 스킨케어 단계 최소화 트렌드
- 클린뷰티 강조
- 미니멀리즘과 품질 중심

#### 프랑스
- 향(香)에 대한 선호도 높음
- 디자인 감성 중요
- 우아함과 세련됨 추구

#### 미국
- 건조한 환경 대응
- 색소 규제 준수
- 대용량 선호
- 지속가능성 중시

#### 동남아
- 고온다습 기후
- 산뜻한 텍스처 필요
- 트로피컬 향 선호

### 기존 K-뷰티가 해결하지 못한 문제

1. **현지화 부족**
   - 국가별 피부환경·기후·생활습관 차이 미반영
   - 대다수 제품이 한국 피부·기후 기준 개발

2. **문화적 이해 부족**
   - 문화적 금기·미용 규제·소비 가치 차이 미반영
   - 한국 기준 제품·마케팅을 단순 번역하여 해외 판매
   - 브랜드 메시지·표현이 현지 가치관과 충돌

3. **글로벌 UX 미흡**
   - K-뷰티는 화려한 디자인과 기획형 패키지 중심
   - 글로벌 표준 UX 조사·현지 소비자 테스트 부족
   - "한국에서 잘 팔리는 디자인"을 그대로 해외 적용

## 3. 솔루션

### Belle Rêve의 4가지 차별화 전략

1. **고객 맞춤형 현지화**
   - 국가별 환경 및 피부 상태에 대한 철저한 사전 조사
   - 기후, 생활습관, 피부 타입 분석

2. **Cultural Identity 반영**
   - 각국의 문화 정체성을 담은 디자인
   - 국가별 선호 향료 선택
   - 문화적으로 의미 있는 용기 디자인

3. **K-뷰티 품질 유지**
   - 검증된 한국 화장품 기술
   - 우수한 품질 관리
   - 국가별 현지 소재 접목

4. **Co-creation 플랫폼**
   - 고객 참여형 제품 개발
   - 커뮤니티 기반 의사결정
   - 높은 초기 충성도 확보

## 4. 목표 시장

### 우선 출시국: 일본

**선정 이유**:
- 뷰티 시장 규모 양호 (2024년 약 320억 달러, 글로벌 시장의 5%)
- K-뷰티 호감도 높음
- 미니멀·클린뷰티·품질 중심 성향과 브랜드 철학 부합

**MVP**: 일본 한정판 올인원 에센스

### 향후 확장 계획
- 2년 차: 프랑스, 미국
- 3년 차: 동남아 및 기타 시장

### 시장 규모 (2024년 기준)
- **글로벌 뷰티 시장**: 약 6,700억 달러
- **일본**: 약 320억 달러
- **프랑스**: 약 450억 유로
- **미국**: 약 430억 달러

## 5. 제품 개발

### Co-creation 5단계 프로세스

#### Step 1: 국가별 초기 유저 리서치
- 오프라인 FGD (Focus Group Discussion)
- Online Community 운영
- 설문조사 실시

#### Step 2: 1차 프로토타입 개발
- 샘플 3종 제작
- 품질 테스트
- 안전성 검증

#### Step 3: 국가별 소비자 테스트
- A/B Testing 실시
- 선호도 조사
- 피드백 수집 및 분석

#### Step 4: 디자인·용기 공동 제작
- 국가별 문화요소 반영
- 커뮤니티 투표
- 최종 디자인 확정

#### Step 5: 크라우드펀딩 + 글로벌 론칭
- 현지 플랫폼 활용 (일본: CAMPFIRE)
- 수요 검증
- 공식 판매 시작

### 단계별 라인업 로드맵

1. **1단계**: 에센스 1종 (MVP)
2. **2단계**: 토너, 크림 추가
3. **3단계**: 클린·비건 라인 확장

## 6. 사업화 전략

### 마케팅 채널

1. **Co-creation 참여자 활용**
   - 무료 제품 제공
   - 제품 테스트 및 리뷰
   - 구전 효과 극대화

2. **크라우드펀딩**
   - 일본: CAMPFIRE 플랫폼
   - 수요 확인 및 초기 판매
   - 브랜드 인지도 구축

3. **인플루언서 마케팅**
   - 국가별 현지 인플루언서 협업
   - 마이크로 KOL 중심
   - SNS 광고 집행

4. **온라인 직구 판매**
   - 자사몰 운영
   - 국가별 최적화된 쇼핑 경험

5. **B2B 유통**
   - 현지 편집숍 입점
   - 리테일 매장 협업
   - 프리미엄 백화점 진출

### 가격 전략
- 프리미엄 에디션: 6~7만원대
- 고품질 포지셔닝
- 가치 기반 가격 책정

## 12. 디자인 시스템

3. **접근성**: 모든 사용자가 쉽게 사용 가능
4. **반응형**: 모바일, 태블릿, 데스크톱 최적화

## 15. 개발 환경 설정

### 필수 요구사항
- Node.js 18.x 이상
- npm 9.x 이상

### 설치 방법

```bash
# 저장소 클론
git clone https://github.com/jera0520/belle-reve.git

# 디렉토리 이동
cd andong-esg-beauty

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 브라우저에서 확인
# http://localhost:3000
```
